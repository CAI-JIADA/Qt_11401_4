# 背景圖片效果說明

## 更新內容 (2026-01-03)

根據使用者需求，背景圖片功能已增強，現在包含以下效果：

### 1. 淡化效果 (Fade Effect)

背景圖片會自動添加 50% 不透明度的白色遮罩層，使背景看起來更柔和，確保前景文字和 UI 元件清晰可讀。

**技術實作：**
```css
background-image: 
    linear-gradient(rgba(255, 255, 255, 0.5), rgba(255, 255, 255, 0.5)),
    url(path/to/image);
```

**效果：**
- 原始圖片色彩會變淡
- 文字對比度提高
- 整體視覺更柔和

### 2. 拉伸至視窗同比例 (Proportional Stretch)

背景圖片會自動縮放以覆蓋整個應用程式視窗，同時保持圖片原始比例。

**技術實作：**
```css
background-size: cover, cover;
background-position: center center;
background-repeat: no-repeat;
```

**效果：**
- 圖片完整覆蓋視窗
- 不會變形或扭曲
- 圖片置中顯示
- 視窗大小改變時自動調整

### 使用建議

1. **圖片選擇**
   - 選擇色彩豐富的圖片效果最佳
   - 高解析度圖片會自動縮放
   - 任何尺寸的圖片都能正常顯示

2. **視覺效果**
   - 淡化效果確保 UI 元件可見
   - 圖片會自動適應不同螢幕尺寸
   - 視窗縮放時背景會保持覆蓋

3. **效能**
   - 圖片只在設定時載入一次
   - 使用 CSS 硬體加速
   - 對效能影響極小

### 技術細節

**CSS 屬性：**
- `background-size: cover` - 縮放圖片以完全覆蓋容器
- `linear-gradient()` - 添加半透明白色遮罩層
- `rgba(255, 255, 255, 0.5)` - 50% 不透明度白色

**Qt 實作：**
- 使用 QWidget StyleSheet
- 支援多層背景疊加
- 自動處理不同圖片比例
