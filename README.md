# Calender

Qt 多平台行事曆整合工具 - 整合 Google、Apple、Outlook 行事曆的待辦事項管理工具

## 專案說明

本專案使用 Qt 框架開發，可以將不同平台的行事曆待辦事項整合到統一的介面中。

### 主要功能

- 🔗 **多平台整合** - 支援 Google Calendar、Apple Calendar、Microsoft Outlook
- 📋 **待辦事項管理** - 整合事件與日程
- 👥 **共享行事曆** - 讀取他人分享的行事曆
- 🔍 **搜索功能** - 快速搜索所有事件
- 📱 **跨裝置支援** - 支援桌面與行動裝置
